# Simple Group Finance Management

## Require:

### Nodejs v18.13.0
### MongoDB v6.0.3
### NPM v8.19.2

## Install:

`npm i -g pnpm`

`pnpm install`

Config env file: Create .env in root folder
Open src\config\devenv.md file and config your own configurations
Then...

`pnpm run dev`
